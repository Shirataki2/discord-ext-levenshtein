Welcome to discord-ext-levenshtein!
===================================================

discord-ext-levenshtein is a simple extension of
`discord.py <https://discordpy.readthedocs.io/ja/latest/>`_.

This is an extension that suggests a
command when the invoked command cannot
be found, based on the Levenshtein distance
from the command registered.
